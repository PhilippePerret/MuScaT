With this script you can activate an analysis from the `./_analyses_` folder.

USAGE
-----
  #{'./analyse.rb "<folder [beginning] name>"'.jaune}

  Note: if you're in the main folder, you must run
  `./utils/analyse.rb`.

  (with `mus` alias) > mus analysis "<folder name>"

  The '<folder [beginning] name>' is the name or the beginning of the
  folder name in the `_analyses_` folder which contains all of the
  analysis. If several folder share the same beginning, the command
  lets you choose one of them.

  If no arg is supplied, the whole analysis list is displayed,
  letting you to choose one of them.
